Vamos a iniciar con la primera feature de este proyecto que es la conexión de la Cuenta de WhatsApp con Notify. 
Usa la skill /automate-whatsapp para conocer como es el proceso de integración de WhatsApp con Kapso. Además puedes encontrar la documetnación aqui https://docs.kapso.ai/docs

El objetivo es registrar las cuentas de whatsapp a nivel de organización, es decir que lo debe hacer un usuario con rol owuner o admin.
