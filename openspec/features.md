## Contactos
1. Debe haber una búsqueda de contactos que permita buscar contactos por nombre y por número de teléfono.

2. En el inbox, debe haber un icono que permita iniciar una nueva conversación con un contacto que no tiene conversación, puede ser un icono a la dercha de la busqueda o un botón en el estado vacio del listado de conversaciones, cuando la busqueda no arroje resultados. Se espera que pueda buscar en el listado de contactos y enviar un mensaje de plantilla. 

3. Se debe poder enviar mensajes de audio directamente, con un icono de grabación, tal como lo permite hoy WhatsApp. En el caso de whatsapp mientras el input de texto está vacío se ve el icono del microfono para iniciar la grabación de audio, una vez inicio a escribir, el microfono se oculta y aparece el botón para enviar el texto.

4. El input del mensaje de texto debe permitir multiples lineas. Puede ser con control + enter o con shift + enter que permita saldo te línea. 

5. Al enviar documentos, imagenes, etc debe permitir el caption. No enviarlas de inmediato, adjuntarlas al chat

6. Para el envío de imagenes debe ser posible pegar directamente desde el portapapeles. 

7. Para el envío de imagenes y documentos debe permitir arrastrar el archivo al área de chat para adjuntarlos. No se deben enviar inmediatamente, pues el texto del input será como el caption.
